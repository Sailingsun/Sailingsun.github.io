san框架使用：

1、注意双向绑定，，否则不会动态变化例如dialog弹窗只能开关一次。页面不显示数据可能是css绝对定位或者层级问题，也可能是js的数据还没到达问题

2、filters中最好不要传true/false等字段

3、有些组件重构性不是很友好：

dialog中的直接关闭

onCloseDialog() {

​ this.data.set(‘open’, false);

​ this.fire(‘close’);

}

先关闭再fire出可扩展的close方法导致无法在关闭前进行某些操作，或进行判断后再关闭

组件扩展常用s-if和slot

4、有些扩展性不是很友好：select组件的搜索不能扩展，只能静态搜索。
